## Sprites

Προτού μπορέσετε να ξεκινήσετε την κωδικοποίηση, θα πρέπει να προσθέσετε ένα «πράγμα» στον κώδικα. Στο Scratch, αυτά τα 'πράγματα' ονομάζονται **sprites**.

+ Αρχικά, ανοίξτε το πρόγραμμα επεξεργασίας Scratch. Μπορείτε να βρείτε τον ηλεκτρονικό επεξεργαστή Scratch στο <a href="http://jumpto.cc/scratch-new" target="_blank">jumpto.cc/scratch-new</a>. Μοιάζει με αυτό:
    
    ![screenshot](images/band-scratch.png)

+ Ο sprite γάτα που μπορείτε να δείτε είναι η μασκότ Scratch. Ας απαλλαγούμε από αυτό κάνοντας δεξί κλικ και μετά πατώντας **διαγράψτε**.
    
    ![screenshot](images/band-delete.png)

+ Στη συνέχεια, κάντε κλικ στο **Επιλέξτε Sprite από τη βιβλιοθήκη** για να ανοίξετε μια λίστα με όλους τους Sprites Scratch.
    
    ![screenshot](images/band-sprite-library.png)

+ Πραγματοποιήστε κύλιση προς τα κάτω έως ότου δείτε ένα σπάιλο τύμπανο Κάντε κλικ σε ένα τύμπανο και κάντε κλικ στο **OK** για να το προσθέσετε στο έργο σας.
    
    ![screenshot](images/band-sprite-drum.png)

+ Κάντε κλικ στο **συρρικνωμένο εικονίδιο** και στη συνέχεια κάντε κλικ στο τύμπανο μερικές φορές για να το κάνετε μικρότερο.
    
    ![screenshot](images/band-shrink.png)

Δώστε στο πρόγραμμα ένα όνομα πληκτρολογώντας ένα στο πλαίσιο κειμένου στην επάνω αριστερή γωνία.

Στη συνέχεια, μπορείτε να κάνετε κλικ στο **Αρχείο** και, στη συνέχεια, **Αποθήκευση τώρα** για να αποθηκεύσετε το έργο σας. Αν δεν έχετε λογαριασμό Scratch, μπορείτε να αποθηκεύσετε ένα αντίγραφο του έργου σας κάνοντας κλικ στο **Λήψη στον υπολογιστή σας** αντί.

![screenshot](images/band-save.png)
